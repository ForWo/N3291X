This filder contains the GVE 2D Graphic Engine (2DGE) low-level library for UI acceleration. 

The major function in 2DGE is BitBLT and Bresenham. 

The BitBLT supports ROP3 (Pattern, Source, Destination and some controls such as color expansion, 
transparent, stretch, tilt, alpha blending and clipping.

The sample programs must run on Nuvoton N3291x DEV board with 800x480 LCM or user needs to modify
the sample code for the hardware change.
